# Workout Set Counter App – Idea

## แนวคิดหลัก
แอพสำหรับการออกกำลังกายที่ช่วยนับเซ็ท นับครั้ง และจัดการเวลาพักแบบอัตโนมัติ  
เน้นใช้งานง่าย ไม่รก ผู้ใช้โฟกัสที่การออกกำลังกายได้เต็มที่

---

## Core Features

### 1. เพิ่มท่าออกกำลังกาย
ผู้ใช้สามารถกำหนด:
- ชื่อท่าออกกำลังกาย
- จำนวนเซ็ท
- จำนวนครั้งต่อเซ็ท (Reps)
- เวลาพัก (วินาที)

มี Preset ให้เลือก:
- Beginner: 3x10 พัก 60 วินาที
- Strength: 5x5 พัก 180 วินาที

---

### 2. หน้านับเซ็ท (Workout Screen)
แสดงข้อมูล:
- ชื่อท่า
- เซ็ทปัจจุบัน (เช่น Set 2 / 4)
- จำนวนครั้ง (Reps)
- ปุ่ม “เสร็จแล้ว”

เมื่อกดปุ่ม:
- เพิ่มเซ็ทอัตโนมัติ
- เข้าสู่โหมดพัก
- เริ่มนับเวลาพักทันที

---

### 3. ระบบพัก (Rest Timer)
- Countdown Timer
- ปุ่ม +10 / -10 วินาที
- ปุ่มข้ามพัก (Skip Rest)
- แจ้งเตือนเมื่อพักครบ (เสียง / สั่น / Notification)

---

### 4. เมื่อทำครบทุกเซ็ท
- แสดงสถานะ Completed
- ตัวเลือก:
  - เล่นท่าเดิมใหม่
  - ไปท่าถัดไป
  - จบการออกกำลังกาย

---

## Features เสริม

### Workout Summary
- จำนวนท่าที่เล่น
- จำนวนเซ็ทรวม
- เวลาที่ใช้ทั้งหมด
- เวลาพักรวม

### History / Log
- ประวัติการออกกำลังกาย
- ดูการพัฒนาของแต่ละท่า

### Smart Rest (อนาคต)
- ปรับเวลาพักอัตโนมัติตามความหนักของเซ็ท
- แนะนำพักเพิ่มในเซ็ทท้าย

---

## User Flow
1. เปิดแอพ
2. เพิ่มท่าออกกำลังกาย
3. เริ่มออกกำลังกาย
4. ทำเซ็ท → กดเสร็จ
5. พักอัตโนมัติ
6. ทำจนครบเซ็ท
7. ดูสรุปผล
